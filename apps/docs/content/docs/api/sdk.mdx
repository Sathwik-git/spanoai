---

The SDK is a thin, typed wrapper over the HTTP + WebSocket API with retries, reconnect,
and idempotent operations built in.

```bash
bun add @spanoai/sdk
```

## Client

```ts
import { SpanoAIClient } from "@spanoai/sdk";

const client = new SpanoAIClient({
  baseUrl: "http://localhost:8000",
  apiKey: process.env.SPANOAI_KEY!,
  agent: "planner", // identity sent on every request
});
```

Every method below is reachable as `client.context.*`, `client.bus.*`,
`client.sessions.*`, or `client.artifacts.*`.

## stream

```ts
const stream = await client.stream(session, (event) => {
  // event.type is "message" | "context" | …
  console.log(event);
});
// later: stream.close();
```

`stream()` mints a single-use [WebSocket ticket](/docs/authentication#websocket-tickets),
connects, and transparently reconnects with gap recovery so you don't miss events
across a blip.

## Errors

Failed requests throw `SpanoAIError` with `status`, `code`, and `requestId`:

```ts
import { SpanoAIError } from "@spanoai/sdk";

try {
  await client.context.write(session, "secrets", "k", { v: 1 });
} catch (e) {
  if (e instanceof SpanoAIError && e.status === 403) {
    // key lacks scope or the namespace isn't on its allowlist
  }
}
```
